
# 📱 User Directory App (React Native + TypeScript)

## 🚀 Overview

This is a React Native mobile application that fetches user data from an external API and displays it in a clean, searchable list. Users can tap on any user to view detailed information.

---

## ✨ Features

* 📡 Fetch users from API
* 📋 Display users using FlatList
* 🔍 Search users by name and email
* 📄 User detail screen
* 🔄 Pull-to-refresh support
* 🎨 Modern card-based UI
* ⚡ Built with TypeScript

---

## 🛠️ Tech Stack

* React Native (Expo)
* TypeScript
* React Navigation
* Hooks (useState, useEffect)

---

## 🌐 API Used

https://jsonplaceholder.typicode.com/users
